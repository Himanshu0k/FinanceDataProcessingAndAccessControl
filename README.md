# 💰 Finance Data Processing and Access Control System

A backend system designed to **process financial data securely** while enforcing **role-based access control (RBAC)**.  
This project focuses on **data validation, structured processing, and controlled access to financial records**.

---

## 🚀 Overview

This project is a **Node.js-based backend application** that:

- Processes financial records
- Validates and normalizes input data
- Applies strict access control rules
- Ensures secure and structured data handling
- Provides test coverage for reliability

It is designed to simulate a **real-world financial backend system** where multiple users interact with sensitive data under controlled permissions.

---

## 🧱 Tech Stack

| Layer        | Technology |
|-------------|-----------|
| Backend      | Node.js, Express.js |
| Database     | PostgreSQL (via Prisma ORM) |
| Testing      | Jest |
| ORM          | Prisma |
| Language     | JavaScript |
